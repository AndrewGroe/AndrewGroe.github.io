Pocket Trivia

Trivia app that pulls questions and categories from opentdb. Backend uses Retrofit and Gson to handle and convert JSON response. Uses Shared Preferences for local storage of API response. Swipe gestures managed with OXSwipe library.

Project Details

  • REST API (Retrofit + Gson)
  • Shared Preferences
  • OXSwipe (Swipe Gestures)
